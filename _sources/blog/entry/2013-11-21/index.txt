Haxe で hello world
================================================================================

Hexe は， Java, JavaScript など， 複数のターゲットにコンパイル できる静的型付き言語だ．
言語仕様をざっと読んだ限り，全体として，OOP の血筋の言語で，個人的には，あまり興味が沸かな
かった．型推論が強力なようで，たとえば，仮引数の型を明示しなくともよいらしく，型システムは，
よく嘲笑の的にされる Java や， C# と比べるとずいぶんと便利だ．しかし，興味が沸かなかった理由
は，ぼくがオッブジェクト指向というものがあまり好きではないからかもしれない．どうしても，データは，
クラスではなく， Algebraic data type のように管理したいと思ってしまう．

そんな `Haxe <http://haxe.org>`_ で hello world してみる．

Haxe のインストールは， Homebrew でできた．拡張子は hx を使うようだ．
class Hello のようにしてクラスをつくり，そのクラスと同名の Hello.hx ファイルに保存する．
hello.hx のように，小文字ではいけない．あるいは，べつの名前をつけてもいけない．

main 関数は static function main() のようにして宣言する．戻り値の型や，
( main 関数の場合は使用しないが ) 仮引数の型は，推論されるので，省略できる．

.. literalinclude:: Hello.hx
   :linenos:

コンパイルは， JavaScript にコンパイルする場合，次のようにする．

.. code-block:: sh

   haxe -main Hello Hello.hx -js hello.js
